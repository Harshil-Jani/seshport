how it works

One slash command. Zero re-explaining.

Every coding agent keeps its transcript on disk in its own format. seshport translates between them, so the next agent picks up exactly where the last one stopped.

step 1

Type /seshport

Mid-conversation, in whichever agent you're using. It ports the current session on the spot.

step 2

Paste the resume command

You get back one line — codex resume <id> or claude --resume <id>. Open the other agent and paste.

step 3

Keep going

Full context travels: decisions, tool output, that codeword you planted three hours ago. Verified with real round-trips.

under the hood

Every agent is five methods away.

Sessions flow through one neutral transcript. Each tool implements a small Tool trait — sniff, import, export — so integrations never touch each other.
